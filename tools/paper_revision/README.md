# Paper Revision Tools

This folder contains standalone scripts used to adjust submission DOCX files outside the main `DSTA-SLR` training code.

## Files

- `address_reviewer_risks.py`: aligns manuscript structure and content against reviewer-risk notes.
- `final_submission_cleanup.py`: applies final DOCX cleanup passes before submission.

## Notes

- These scripts currently point to absolute local DOCX paths.
- They are intentionally separated from `DSTA-SLR/scripts/` because they are paper-editing utilities, not experiment runners.
