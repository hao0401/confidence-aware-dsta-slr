# Confidence-Aware DSTA-SLR

This repository hosts a confidence-aware extension of DSTA-SLR for skeleton-based sign language recognition. The public root is intentionally kept minimal so the main research code, experiment workflows, and release-facing documentation are easy to navigate.

## Start Here

- Main codebase: [`DSTA-SLR/`](DSTA-SLR/)
- Setup and usage guide: [`DSTA-SLR/readme.md`](DSTA-SLR/readme.md)
- Script overview: [`DSTA-SLR/scripts/README.md`](DSTA-SLR/scripts/README.md)
- Experiment runbook: [`DSTA-SLR/scripts/EXPERIMENT_RUNBOOK.md`](DSTA-SLR/scripts/EXPERIMENT_RUNBOOK.md)
- License: [`DSTA-SLR/LICENSE`](DSTA-SLR/LICENSE)

## Repository Scope

The public repository includes the training code, model changes, experiment runners, and reporting utilities needed to reproduce the confidence-aware DSTA-SLR workflows.

Datasets, checkpoints, training outputs, and private paper-revision materials are intentionally excluded from version control.
